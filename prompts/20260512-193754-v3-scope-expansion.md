### Performance target

Single-threaded sequential read throughput — the only metric the
user picked. This narrows the design considerably:

- `memmap2` + `madvise(MADV_SEQUENTIAL)` for cold-cache wins
- Zero-copy `&[u8]` slices through to writers via a `RandomAccess`
  trait
- SIMD CRC-16-CCITT (UDF tag CRC) via CLMUL on x86_64 and PMULL on
  aarch64
- No rayon. Default single-threaded.

Writes need only be correct, not fast. Phase 4 will not be
optimized; reaching fsck-clean output is the bar.
